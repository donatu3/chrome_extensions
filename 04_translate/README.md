# wikisearch
検索中に気になった文字を選択し、右クリックから翻訳を選択するとアラートで訳を出す。
わざわざ翻訳サイトを開いて、コピー＆ペーストしなくても良い点が特徴。
※無料版なので月20万文字まで※
