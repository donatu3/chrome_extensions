# delete history
右上のアイコンをクリックすることでキャッシュの消去を行う。  
誤って削除することがないように一旦ユーザーに確認を取る。  
期間は、全て、１週間前、１日前、１時間前の中から選択可能。  
削除完了通知はバッジに表示する。
