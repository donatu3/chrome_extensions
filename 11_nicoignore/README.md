# nicoignore
ニコニコ動画の検索結果（キーワード、タグ検索）から不要な投稿者の動画を非表示にする。  
オプションから投稿者のユーザIDを登録すれば、検索後に自動で非表示にする。  
検索結果の動画IDを用いて投稿者IDを調べ、除外リストと一致した場合に除外する。
