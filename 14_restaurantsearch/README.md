# restaurantsearch
緯度、経度、距離、価格を設定することでその付近の食べ物屋をランダムに表示する。  
出力内容は、店名、ジャンル、参考価格、営業時間、住所、GoogleMAP住所検索リンク。  
ご飯をどこ食べ行くか悩んだときに利用できる。
